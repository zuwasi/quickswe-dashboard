What is QuickSWE?

🎯 Purpose

  • Fast, rigorous benchmark for AI coding agents
  • 100 tasks across 3 languages, 4 difficulty tiers
  • Bug fixes, features, and refactoring

⚡ Practical

  • No Docker — runs on your laptop
  • Under 100MB (vs 200GB for SWE-Bench Pro)
  • Built-in statistical multi-run analysis

📏 Rigorous

  • Every measurement backed by real pytest
  • No LLM-as-judge, no subjective scoring
  • Tests pass or they don't

🔴 Hard

  • Compilers, VMs, GCs, type inference
  • Lock-free data structures, coroutines
  • CUDA dynamic parallelism, warp BFS, FFT

Methodology

🔬 Test Protocol

  • Agent gets task description + buggy source
  • Must fix code to pass fail_to_pass tests
  • Existing pass_to_pass tests must NOT break
  • Wall-clock time measured per task

📊 Statistical Rigor

  • 3 independent runs per agent
  • Agents alternate order each round
  • Consistency score per task
  • Aggregate resolve rates with variance

🛡️ Safety

  • NTFS deny-delete ACLs on critical dirs
  • Guardrail prompts (CLAUDE.md / AGENTS.md)
  • Post-task directory integrity checks
  • Isolated temp workspaces per task

⚖️ Fairness

  • Same prompt, timeout (600s), machine
  • Both use bypass-permissions mode
  • Latest versions (April 2026)
  • Alternating agent order per round

Key Findings

  1. Amp Deep³ wins across all three languages — Python, C++, and CUDA
  2. Gap widens with complexity: Python (+1%) → CUDA (+16%) → C++ (+46%)
  3. Far fewer regressions: Amp breaks tests 6x less often (2.1% vs 13.3%)
  4. Perfect consistency: Amp resolves the exact same tasks every run on Python & CUDA
  5. Claude hit API rate limits during testing — a reliability concern for intensive workloads
  6. Amp trades speed for accuracy: slower but significantly higher resolve rate on hard tasks
